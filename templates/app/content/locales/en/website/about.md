# About {{appName}}

We believe building software should be fast, fun, and accessible to everyone.

## Our Mission

To provide the best developer experience for building full-stack applications, so you can focus on what matters: your users.

## The Stack

{{appName}} is built with:

- **TypeScript** — End-to-end type safety
- **Hono** — Ultra-fast HTTP server
- **tRPC** — Type-safe API layer
- **Drizzle ORM** — Type-safe database queries
- **React** — Component-based UI
- **Vibe on Rails** — The framework that ties it all together
